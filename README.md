# Avaliações de Atrações Turísticas na Rota das Emoções

Este repositório contém o banco de dados tratado e o script de análise estatística para o estudo sobre a relação entre nota e extensão textual em avaliações online de atrações turísticas na Rota das Emoções (TripAdvisor).

## Conteúdo

- `rota_final.csv`: banco de dados filtrado, com outliers removidos
- `script.R`: código completo em R para limpeza, análise e visualização
- `boxplot_grupo.png`: gráfico de distribuição do tamanho dos comentários por grupo

## Objetivo

Investigar se há diferença significativa no tamanho dos comentários entre os grupos de detratores (notas 1 e 2) e promotores (notas 4 e 5), utilizando testes estatísticos não paramétricos.
